# FishHook Shop

Онлайн-магазин рибальської снаряги (навчальний проєкт).

## Стек

- Backend: Node.js + Express + SQLite
- Frontend: React (Vite)
- Архітектура: модульний моноліт, DDD-орієнтований дизайн
